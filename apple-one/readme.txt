Apple1 for ep4ce6 Dev Board.

Notes: 

The apple 1 project added to here is primarily the work of others, see readme file in doc folder for details.
24Bit VGA dac Module (aliexpress) used to provide VGA display see module pinout jpg file for details.
PS2 Keyboard connected (ps2clk - ps2data - gnd - 3.3v) see project pin assignments for details.

Build:

* Open the boards/ep4ce6 folder and compile apple-one_cyclone4 project file using Quartus.
* Program ep4ce6 dev Board.
