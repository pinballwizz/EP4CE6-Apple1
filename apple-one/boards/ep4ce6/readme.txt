Apple1 for ep4ce6 Dev Board.

Notes: 

24Bit VGA dac Module (aliexpress) used to provide VGA display see pinout jpg file for details.
PS2 Keyboard connected (ps2clk - ps2data - gnd - 3.3v) see project pin assignments for details.

Build:

* Open the ep4ce6 folder and compile apple-one_cyclone4 project file using Quartus.
* Program ep4ce6 dev Board.
